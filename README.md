# API 来源
开放的[API](http://www.wanandroid.com/blog/show/2),借鉴android版的wanAndroid原型，进行浏览器apk开发，后期打包成apk可以运行在各平台，可能采用HBuildX进行打包成APK，进行发布。。。。
# 项目搭建

使用 Vue-cli3.0+Vue+Vuex+Vant(移动UI框架)+vue-router，基础功能搭建
# web端直接用浏览器查看
